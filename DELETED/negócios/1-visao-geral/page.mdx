# Visão Geral do Sistema

import { Callout } from 'nextra/components'

Ao acessar o CRM, a tela principal exibe uma lista de leads (potenciais clientes) organizadas em **grupos** e **origens**.

<img src="/prints/lista-leads.png" style={{ borderRadius: '0.5rem' }} />

## Principais Áreas

- **Leads:** Lista de clientes ou oportunidades.
- **Grupos:** Segmentos ou setores da empresa que podem agrupar leads com características ou origens semelhantes.
- **Origens:** Fontes de onde as leads estão vindo (por exemplo, Instagram, tráfego pago, parcerias).
