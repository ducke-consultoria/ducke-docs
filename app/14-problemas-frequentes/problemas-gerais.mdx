# Problemas Gerais

Aqui você encontrará soluções para os problemas mais comuns relacionados ao uso geral do Ducke CRM.

## Como adicionar um novo contato?

1. Acesse o menu "Contatos"
2. Clique no botão "+ Novo Contato"
3. Preencha os campos obrigatórios (nome e e-mail)
4. Clique em "Salvar"

## Como criar um novo agendamento?

1. Acesse o menu "Agenda"
2. Clique no botão "+ Novo Agendamento"
3. Selecione a data e hora desejada
4. Escolha o contato relacionado
5. Adicione uma descrição
6. Clique em "Salvar"

## Como exportar relatórios?

1. Acesse o menu "Relatórios"
2. Selecione o tipo de relatório desejado
3. Configure os filtros necessários
4. Clique em "Exportar"
5. Escolha o formato desejado (PDF, Excel, etc.)

## Como personalizar campos?

1. Acesse "Configurações"
2. Selecione "Campos Personalizados"
3. Escolha o módulo que deseja personalizar
4. Clique em "+ Novo Campo"
5. Configure o tipo e as propriedades do campo
6. Salve as alterações

## Como gerenciar permissões de usuários?

1. Acesse "Configurações"
2. Selecione "Usuários e Permissões"
3. Escolha o usuário que deseja configurar
4. Marque/desmarque as permissões necessárias
5. Salve as alterações

## Problemas com sincronização de dados

Se você estiver enfrentando problemas com a sincronização de dados:

1. Verifique sua conexão com a internet
2. Limpe o cache do navegador
3. Faça logout e login novamente
4. Se o problema persistir, entre em contato com o suporte

## Como fazer backup dos dados?

1. Acesse "Configurações"
2. Selecione "Backup e Restauração"
3. Escolha "Fazer Backup"
4. Selecione os dados que deseja incluir
5. Clique em "Iniciar Backup"

## Problemas com notificações

Se você não está recebendo notificações:

1. Verifique as configurações de notificação em seu perfil
2. Confirme se o e-mail está correto
3. Verifique a pasta de spam
4. Atualize as configurações de notificação conforme necessário
