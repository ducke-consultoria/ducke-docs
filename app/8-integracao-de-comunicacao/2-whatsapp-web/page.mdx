# WhatsApp Web

import { Callout } from 'nextra/components'

<Callout type="info">
Em breve, você poderá integrar o WhatsApp Web ao Ducke CRM para gerenciar suas conversas diretamente da plataforma.
</Callout>

Estamos trabalhando para trazer esta funcionalidade em breve. Com ela, você poderá:

- Gerenciar conversas do WhatsApp Web
- Responder mensagens diretamente do Ducke CRM
- Manter histórico de conversas
- Integrar com outros recursos do sistema
- Automatizar respostas

Lembre-se de seguir as diretrizes do Whatsapp: https://www.whatsapp.com/legal/business-terms/

Fique atento às atualizações para saber quando esta funcionalidade estará disponível.
