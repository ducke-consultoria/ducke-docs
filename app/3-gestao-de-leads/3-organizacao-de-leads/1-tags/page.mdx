# Uso de Tags em Leads

Tags são etiquetas que ajudam a classificar e organizar seus leads de forma personalizada. Este guia irá te mostrar como usar tags no Ducke CRM.

## O que são Tags?

Tags são:
- Etiquetas personalizadas cadastradas em **Configuração → Tags**
- Forma de classificação adicional além do estágio
- Filtros visuais no Kanban
- Marcadores de características específicas

Antes de aplicar tags nos leads, cadastre-as na [configuração de tags](/10-configuracao-do-sistema/0-configuracao-de-tags).

## Como Adicionar Tags a um Lead

1. Acesse o lead desejado no quadro ou na lista
2. Clique no ícone de tag no card ou na ficha do lead
3. Selecione uma tag existente ou digite o nome de uma nova (se tiver permissão)
4. Confirme a seleção

## Exemplos de Uso

- Setor da empresa
- Tipo de produto/serviço
- Prioridade
- Fonte de contato
- Status específico

## Dicas de Uso

- Use tags consistentes — padronize na tela de configuração
- Mantenha uma lista enxuta de etiquetas ativas
- Combine com estágios e filtros para segmentar follow-ups
- Use cores distintas para identificar categorias rapidamente no quadro

As tags são reutilizáveis e podem ser usadas em múltiplos leads.

## Próximos Passos

Após adicionar tags, você pode:
- Filtrar leads por tag no quadro
- Criar relatórios segmentados
- Organizar visualmente o funil
- Identificar padrões de conversão por segmento
