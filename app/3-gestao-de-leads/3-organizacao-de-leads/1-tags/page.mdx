# Uso de Tags em Leads

Tags são etiquetas que ajudam a classificar e organizar seus leads de forma personalizada. Este guia irá te mostrar como usar tags no Ducke CRM.

## O que são Tags?

Tags são:
- Etiquetas personalizadas
- Forma de classificação adicional
- Filtros visuais no Kanban
- Marcadores de características específicas

## Como Adicionar Tags

1. Acesse o lead desejado
2. Clique no ícone de tag
3. Digite o nome da tag
4. Pressione Enter para confirmar

## Exemplos de Uso

- Setor da empresa
- Tipo de produto/serviço
- Prioridade
- Fonte de contato
- Status específico

## Dicas de Uso

- Use tags consistentes
- Mantenha uma lista padrão
- Combine com estágios
- Use para filtros rápidos

:::tip
As tags são reutilizáveis e podem ser usadas em múltiplos leads.
:::
