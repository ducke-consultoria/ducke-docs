# Classificação de Leads

A classificação de leads ajuda a organizar e priorizar seus contatos. Este guia irá te mostrar como classificar leads no Ducke CRM.

## Métodos de Classificação

1. **Por Estágio**
   - Novos
   - Em negociação
   - Em análise
   - Fechados

2. **Por Status**
   - Abertos
   - Ganhos
   - Perdidos

3. **Por Tags**
   - Setor
   - Prioridade
   - Tipo de produto

## Como Classificar

1. Use o Kanban para classificação visual
2. Aplique tags para detalhamento
3. Utilize o status para finalização
4. Adicione notas para contexto

## Dicas de Classificação

- Mantenha padrões consistentes
- Use múltiplos critérios
- Documente decisões
- Revise periodicamente

:::tip
Combine diferentes métodos de classificação para uma organização mais eficiente.
:::
