# Gestão de Leads Ganhos

Leads ganhos são aqueles que se tornaram clientes ou fecharam negócio. Este guia irá te mostrar como gerenciar esses leads.

## Como Marcar um Lead como Ganho

1. Acesse o lead desejado
2. Clique no status do lead
3. Selecione "Ganho"
4. Confirme a alteração

## O que Acontece?

- O lead ficará verde no sistema
- Poderá ser movido para uma área específica de clientes
- Permanecerá no histórico para consulta

## Dicas de Gestão

- Registre o valor do negócio fechado
- Adicione notas sobre o processo
- Mantenha informações importantes
- Crie agendamentos para pós-venda

:::tip
Use o histórico para analisar o que funcionou bem no processo de vendas.
:::

## Próximos Passos

Após marcar um lead como ganho:
- Inicie o processo de pós-venda
- Crie agendamentos de acompanhamento
- Adicione notas de sucesso
- Prepare para próximas oportunidades
