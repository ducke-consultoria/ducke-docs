# Criação Manual de Lead

Criar leads manualmente no Ducke CRM é um processo simples e direto. Este guia irá te mostrar como adicionar novos leads ao sistema.

## Passo a Passo

1. Acesse a seção de Negócios
2. Clique em "Adicionar Negócio" ou no botão de negócio no canto superior direito
3. Preencha os campos básicos:
   - Nome do negócio
   - Telefone
   - Estágio (etapa do funil)
4. Clique em "Salvar"

## Campos Adicionais

Após criar o lead, você pode adicionar mais informações:
- Nome secundário
- CNPJ/CPF
- Endereço
- Segmento
- Quantidade de funcionários
- Faturamento

## Dicas

- O telefone é obrigatório para criar um lead
- Você pode adicionar múltiplos contatos para o mesmo lead
- O sistema criará automaticamente um contato com o telefone informado

:::tip
Use o nome secundário para armazenar informações adicionais como razão social ou nome fantasia.
:::
