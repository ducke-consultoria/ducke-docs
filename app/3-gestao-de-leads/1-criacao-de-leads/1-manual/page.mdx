# Criação Manual de Lead

Criar leads manualmente no Ducke CRM é um processo simples e direto. Este guia mostra como adicionar uma nova oportunidade pelo quadro.

<img src="/prints/leads-criar-oportunidade.png" alt="Modal de criação de oportunidade" style={{ borderRadius: '0.5rem' }} />

## Passo a passo

1. Acesse **Leads** no menu principal.
2. Na aba **Quadro**, clique em **Adicionar Oportunidade**.
3. Preencha os campos básicos:
   - **Nome do Oportunidade**
   - **Telefone**
4. Clique em **Criar Lead**.

## Campos adicionais

Após criar o lead, abra a ficha do card para incluir:

- Nome secundário
- CNPJ/CPF
- Endereço
- Segmento
- Quantidade de funcionários
- Faturamento

## Dicas

- O telefone é obrigatório para criar um lead.
- Você pode adicionar múltiplos contatos para o mesmo lead.
- O sistema cria automaticamente um contato com o telefone informado.

Use o nome secundário para informações adicionais, como razão social ou nome fantasia.

## Próximos passos

Após criar o lead, você pode:

- Adicionar mais contatos
- Criar agendamentos
- Adicionar notas
- Anexar arquivos
