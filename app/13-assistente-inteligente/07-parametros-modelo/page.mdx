# Ajustar Parâmetros do Agente

1. Com o agente selecionado, abra **Configurações do Agente** no painel direito.
2. **Modelo**: escolha entre opções (Nano, GPT-3.5, GPT-4) conforme custo, velocidade e precisão.

   * **Nano**: até 5M tokens, custo mais baixo, respostas objetivas.
   * **GPT-3.5**: até 1.5M tokens, equilíbrio entre custo e performance.
   * **GPT-4**: até 100k tokens, maior profundidade e qualidade.
3. **Raciocínio vs Inteligência**:

   * Inteligência: respostas diretas com base no script.
   * Raciocínio: reflexão interna antes de responder, ótimo para casos complexos.
4. **Max Tokens**: limite de tokens por resposta (define comprimento máximo).
5. **Temperatura** (apenas em modelos de Inteligência): controla aleatoriedade (0.01 = rigoroso; 0.5–1 = criativo).
6. Monitore o consumo em **Relatórios** para ajustar o uso conforme orçamento.
