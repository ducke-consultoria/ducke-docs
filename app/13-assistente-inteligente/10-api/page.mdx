# Configurar Bloco API

1. Arraste o bloco **API** para o fluxograma.
2. Clique no bloco e preencha os campos:

   * **URL**: endpoint externo.
   * **Headers**: chaves e tokens de autenticação.
   * **Body**: payload em JSON usando `{{variavel}}`.
3. Escolha o método HTTP (GET, POST, PUT, DELETE).
4. Salve e clique em **Testar** para enviar uma requisição sem afetar produção (use **Ambiente de Teste**).
5. Verifique a resposta na aba **Logs** e ajuste conforme necessário.
