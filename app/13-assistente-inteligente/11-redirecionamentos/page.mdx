# Criar Redirecionamentos Condicionais

1. Conecte blocos para definir caminhos alternativos.
2. Em **Seguir quando...**, configure condições lógicas ou expressões regex.
3. Exemplos:

   * Se cliente disser "agendar", redirecione para bloco de agendamento.
   * Se disser "comprar", redirecione para bloco de vendas.
4. Use o bloco **Condição** para criar múltiplas saídas de forma clara.
