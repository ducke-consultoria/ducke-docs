# Criar um Novo Assistente

1. No menu, clique em **Assistentes** e depois em **Criar Assistente**.
2. Preencha **Nome** (identificação interna) e **Descrição** (texto visível ao cliente).
3. Abra o **fluxograma** clicando no ícone correspondente para exibir o nó **Início**.
4. Defina a **Origem** (ex.: WhatsApp, Instagram) que ativará este assistente.
5. Clique em **Salvar** para registrar o novo assistente.
6. Use o botão **Testar** para enviar uma mensagem de exemplo e validar o disparo do fluxo.
7. Acesse **Logs** para verificar as requisições recebidas e as respostas geradas pelo assistente.
