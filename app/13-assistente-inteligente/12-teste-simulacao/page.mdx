# Testar e Simular o Assistente

1. Selecione **Ambiente de Teste** no topo do editor para não afetar dados reais.
2. Use o botão **Simular** (ícone de ▶️) para enviar mensagens de teste via WhatsApp de desenvolvimento.
3. Confira na aba **Logs** as requisições, variáveis capturadas e respostas geradas.
4. No histórico do CRM, confira tags automáticas (ex.: **Necessita de Ação**) e o ícone de handoff.
5. Para ativar/desativar o agente, clique na bolinha de status (verde/vermelha) ao lado do nome.
