# Configurar Bloco Agente

1. Arraste o bloco **Agente** para o fluxograma, próximo ao nó **Início** ou a outro bloco.
2. Clique no agente para abrir as configurações.
3. Defina o **Nome** (visível no editor) e insira o **Script de Instruções** (texto que orienta a IA).
4. Use variáveis no script com a sintaxe `{{nome_da_variavel}}` para inserir dados capturados.
5. Adicione exemplos de perguntas e respostas para treinar o comportamento do agente.
6. Salve e teste o agente no ambiente de simulação para verificar o fluxo.
7. Para remover, selecione o bloco e pressione **Delete**.
