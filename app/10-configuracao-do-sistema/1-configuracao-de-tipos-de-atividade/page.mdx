# Configuração de Tipos de Atividade

import { Callout } from 'nextra/components'

No Ducke CRM, você pode personalizar os tipos de atividades para melhor se adequar ao seu fluxo de trabalho. Esta seção explica como configurar e gerenciar os diferentes tipos de atividades disponíveis no sistema.

## Tipos de Atividade Disponíveis

O sistema oferece os seguintes tipos de atividades por padrão:

- **Ligação**: Para registrar chamadas telefônicas
- **Reunião**: Para agendar e acompanhar reuniões
- **E-mail**: Para registrar comunicações por e-mail
- **Tarefa**: Para acompanhar tarefas pendentes
- **Visita**: Para registrar visitas a clientes
- **WhatsApp**: Para registrar interações via WhatsApp

## Como Configurar um Tipo de Atividade

1. Acesse **Configurações > Tipos de Atividade**
2. Clique em **Novo Tipo de Atividade**
3. Preencha os campos:
   - **Nome**: Nome do tipo de atividade
   - **Descrição**: Breve descrição do propósito
   - **Cor**: Escolha uma cor para identificação visual
   - **Ícone**: Selecione um ícone representativo
   - **Duração Padrão**: Tempo médio estimado
4. Clique em **Salvar**

## Campos Personalizados

Para cada tipo de atividade, você pode adicionar campos personalizados:

1. Selecione o tipo de atividade
2. Clique em **Campos Personalizados**
3. Adicione novos campos:
   - **Tipo**: Texto, Número, Data, Seleção, etc.
   - **Nome**: Nome do campo
   - **Obrigatório**: Se o campo é obrigatório
   - **Valores**: Para campos de seleção

<Callout type="info">
Você pode desativar tipos de atividade que não são utilizados em seu fluxo de trabalho, mantendo apenas os relevantes para sua operação.
</Callout>

## Dicas de Configuração

- Mantenha os nomes dos tipos de atividade claros e objetivos
- Use cores diferentes para facilitar a identificação visual
- Configure campos obrigatórios apenas quando realmente necessário
- Revise periodicamente os tipos de atividade para manter apenas os relevantes
