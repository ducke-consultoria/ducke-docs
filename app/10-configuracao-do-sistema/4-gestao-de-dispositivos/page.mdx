# Gestão de Dispositivos

import { Callout } from 'nextra/components'

O Ducke CRM oferece recursos para gerenciar dispositivos conectados ao sistema, garantindo segurança e controle sobre o acesso à plataforma.

## Registro de Dispositivos

### Como Registrar um Dispositivo

1. Acesse **Configurações > Dispositivos**
2. Clique em **Registrar Novo Dispositivo**
3. Preencha as informações:
   - **Nome do Dispositivo**: Identificação do dispositivo
   - **Tipo**: Desktop, Mobile, Tablet
   - **Sistema Operacional**: Windows, macOS, Android, iOS
   - **Localização**: Local de uso do dispositivo
   - **Usuário**: Responsável pelo dispositivo

### Configurações de Segurança

Para cada dispositivo, você pode configurar:
- **Autenticação de Dois Fatores**
- **Bloqueio Remoto**
- **Limite de Tentativas de Login**
- **Horários de Acesso**
- **Restrições de IP**

## Monitoramento de Dispositivos

### Status dos Dispositivos

O sistema mostra:
- **Último Acesso**
- **Status de Conexão**
- **Atividades Recentes**
- **Alertas de Segurança**
- **Uso de Recursos**

### Relatórios de Uso

Gere relatórios sobre:
- **Tempo de Uso**
- **Aplicações Acessadas**
- **Dados Consumidos**
- **Incidentes de Segurança**
- **Tentativas de Acesso**

## Políticas de Segurança

### Configuração de Políticas

1. Acesse **Configurações > Políticas de Segurança**
2. Configure:
   - **Requisitos de Senha**
   - **Frequência de Troca**
   - **Bloqueio Automático**
   - **Backup de Dados**
   - **Criptografia**

### Aplicação de Políticas

As políticas podem ser aplicadas:
- **Globalmente**: Para todos os dispositivos
- **Por Grupo**: Para grupos específicos
- **Individualmente**: Para dispositivos específicos

<Callout type="info">
Mantenha suas políticas de segurança atualizadas e revise regularmente o acesso dos dispositivos ao sistema.
</Callout>

## Dicas de Gestão

- Mantenha um inventário atualizado dos dispositivos
- Implemente políticas de segurança adequadas
- Monitore regularmente o acesso
- Treine usuários sobre boas práticas
- Faça backup regular dos dados
- Revise logs de acesso periodicamente
