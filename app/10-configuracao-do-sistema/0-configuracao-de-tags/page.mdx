# Configuração de Tags

As **tags** são etiquetas reutilizáveis que classificam leads em todo o funil. Antes de usá-las nos cards do quadro, cadastre-as em **Configuração → Tags**.

<img src="/prints/setup-tags.png" alt="Tela de configuração de tags no setup" style={{ borderRadius: '0.5rem' }} />

## Como acessar

1. Abra **Configuração** no menu (ícone de engrenagem).
2. Selecione a aba **Tags**.
3. Gerencie nome e cor de cada etiqueta.

## Criar uma tag

1. Clique em **Nova tag** (ou equivalente na barra).
2. Informe o **nome** — use termos curtos e padronizados (ex.: *VIP*, *Retorno*, *Indicação*).
3. Escolha uma **cor** para identificação visual no quadro.
4. Salve.

## Editar ou excluir

- Use os ícones de ação em cada linha da tabela.
- A exclusão remove a tag da operação; leads que a usavam deixam de exibi-la.

## Permissões

Apenas usuários com permissão de **Configuração → Tags** podem criar ou alterar etiquetas. Consulte o administrador se a aba não aparecer.

## Uso nos leads

Após cadastrar tags aqui, aplique-as nos cards do funil ou na ficha do lead. Veja também [Uso de Tags em Leads](/3-gestao-de-leads/3-organizacao-de-leads/1-tags).
