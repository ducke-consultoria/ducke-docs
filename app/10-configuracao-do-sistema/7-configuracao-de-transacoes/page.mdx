# Histórico de Transações

A aba **Transações** mostra o consumo de créditos da operação — envios, assistentes, campanhas e outros eventos cobrados pelo sistema.

<img src="/prints/setup-transactions.png" alt="Tela de transações no setup" style={{ borderRadius: '0.5rem' }} />

## Como acessar

1. Abra **Configuração** no menu.
2. Selecione a aba **Transações**.

## Ler a tabela

Cada linha registra:

| Coluna | Significado |
|--------|-------------|
| **Data** | Quando a transação ocorreu |
| **Tipo** | Origem do consumo (ex.: assistente, mensagem) |
| **Créditos** | Valor debitado (vermelho) ou creditado (verde) |

Valores negativos indicam gasto; positivos, recarga ou estorno.

## Detalhes de assistente

Para transações ligadas a assistentes, o ícone de informação abre um popover com mensagens e nós do fluxo envolvidos — útil para auditar consumo automatizado.

## Atualizar a lista

Use o botão de **atualizar** (ícone circular) no canto superior direito para recarregar o histórico mais recente.

## Quando usar

- Conferir gasto diário ou mensal de créditos.
- Investigar picos após campanhas ou fluxos de assistente.
- Validar se recargas de pacote foram aplicadas.
