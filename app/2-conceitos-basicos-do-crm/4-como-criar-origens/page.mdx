# Como Criar Origens

import { Callout } from 'nextra/components'

Criar origens no Ducke CRM permite rastrear de onde vêm seus leads. Siga este guia para configurar suas origens:

## Passo a Passo para Criar Origens

1. Acesse o menu "Configurações"
2. Selecione "Origens" no menu lateral
3. Clique no botão "Nova Origem"
4. Preencha os campos:
   - Nome da Origem
   - Descrição (opcional)
   - Tipo de Origem
   - Cor de identificação
5. Clique em "Salvar"

## Dicas para Criação de Origens

- Crie origens específicas para cada canal de captação
- Mantenha um padrão de nomenclatura
- Inclua origens para todos os seus canais de marketing
- Atualize as origens conforme novos canais são adicionados

<Callout type="info">
  Diferente dos grupos, você pode adicionar múltiplas origens a um lead, permitindo um rastreamento mais detalhado de como ele chegou até você.
</Callout>
