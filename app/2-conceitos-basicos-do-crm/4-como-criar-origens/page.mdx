# Como Criar Origens

import { Callout } from 'nextra/components'

Criar origens no Ducke CRM permite rastrear de onde vêm seus leads. Siga este guia para configurar suas origens:

## Passo a Passo para Criar Origens

1. Na lateral esquerda da tela, posicione o mouse em cima do grupo que você deseja criar a origem. 
2. Clique no ´´+´´ que aparecerá ao lado do nome do grupo
3. Preencha os campos:
   - Nome da Origem
4. Clique em ´´Confirmar´´
5. Para editar o sua origem, siga o seguinte passo a passo:
   - Posicione o mouse em cima do nome da origem que deseja editar
   - Clique no lápis que aparecerá ao lado do nome do grupo
   - Realize as alterações necessárias (nome e permissões)
6. Clique em ´´Salvar´´

## Dicas para Criação de Origens

- Crie origens específicas para cada canal de captação
- Mantenha um padrão de nomenclatura
- Inclua origens para todos os seus canais de marketing
- Atualize as origens conforme novos canais são adicionados

<Callout type="info">
  Diferente dos grupos, você pode adicionar múltiplas origens a um lead, permitindo um rastreamento mais detalhado de como ele chegou até você.
</Callout>
