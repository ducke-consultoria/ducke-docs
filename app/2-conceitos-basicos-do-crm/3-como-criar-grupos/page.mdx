# Como Criar Grupos

import { Callout } from 'nextra/components'

Criar grupos no Ducke CRM é um processo simples que ajuda a organizar seus leads de forma eficiente. Siga estes passos:

## Passo a Passo para Criar Grupos

1. Acesse o menu "Configurações"
2. Selecione "Grupos" no menu lateral
3. Clique no botão "Novo Grupo"
4. Preencha os campos:
   - Nome do Grupo
   - Descrição (opcional)
   - Cor de identificação
5. Clique em "Salvar"

## Dicas para Criação de Grupos

- Use nomes claros e objetivos
- Mantenha uma estrutura hierárquica simples
- Evite criar muitos grupos desnecessários
- Considere o fluxo de trabalho da sua equipe

<Callout type="info">
  Lembre-se que um lead só pode pertencer a um grupo por vez, então planeje sua estrutura de grupos com cuidado.
</Callout>
