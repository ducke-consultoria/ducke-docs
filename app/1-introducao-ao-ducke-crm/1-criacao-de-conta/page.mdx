# Criação de Conta

Para começar a usar o Ducke CRM, o primeiro passo é criar sua conta. Este processo é simples e rápido:

1. Acesse o link do Ducke CRM
2. Clique no botão "Cadastrar" localizado no canto superior direito
3. Preencha os campos necessários:
   - Nome
   - E-mail
   - Senha
4. Clique em "Criar Conta"

Após a criação da conta, você será direcionado para a tela de configuração de operação, onde poderá começar a personalizar seu ambiente de trabalho.

:::tip
Na base de teste (dev), você pode usar qualquer e-mail e senha para criar sua conta.
:::
