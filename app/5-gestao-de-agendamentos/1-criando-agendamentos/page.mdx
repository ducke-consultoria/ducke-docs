# Criando Agendamentos

O Ducke CRM oferece uma interface intuitiva para criar e gerenciar agendamentos de forma eficiente.

## Como Criar um Agendamento

1. Acesse a seção de Operações
2. Clique em "Agenda"
3. Selecione o dia do agendamento e preencha os detalhes do agendamento:
   - Usuário
   - Tipo de atividade
   - Duração
   - Horário do agendamento
4. Clique em "Agendar"

## Campos Obrigatórios

- Título do agendamento
- Data e hora
- Tipo de atividade
- Pelo menos um usuário responsável


Use descrições claras e objetivas para facilitar o entendimento do propósito do agendamento.
