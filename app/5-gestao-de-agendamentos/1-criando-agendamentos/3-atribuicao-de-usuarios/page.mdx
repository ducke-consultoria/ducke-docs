# Atribuição de Usuários

Gerencie a participação de usuários nos agendamentos de forma eficiente.

## Tipos de Participação

- Organizador
- Participante
- Convidado
- Observador

## Como Atribuir Usuários

1. Acesse o agendamento
2. Clique em "Adicionar Participantes"
3. Selecione os usuários desejados
4. Defina o tipo de participação
5. Clique em "Confirmar"

## Permissões por Tipo de Participação

### Organizador
- Editar agendamento
- Adicionar/remover participantes
- Alterar status
- Cancelar agendamento

### Participante
- Visualizar detalhes
- Confirmar presença
- Adicionar observações
- Sugerir alterações

### Convidado
- Visualizar detalhes básicos
- Confirmar presença
- Receber notificações

### Observador
- Apenas visualização
- Sem interação direta


Atribua as permissões adequadas a cada participante para manter a organização e segurança dos agendamentos.
