# Tags de Contatos

As tags são uma forma eficiente de organizar e categorizar seus contatos. Este guia irá te mostrar como usar tags para melhorar a gestão de contatos.

## O que são Tags?

Tags são:
- Marcadores personalizados
- Categorias de contatos
- Filtros de busca
- Organizadores visuais

## Como Usar Tags

1. Acesse um contato
2. Clique no ícone de tags
3. Selecione tags existentes ou crie novas
4. Salve as alterações

## Tipos de Tags

Você pode criar tags para:
- Função (ex: Decisor, Influenciador)
- Área (ex: Compras, TI, RH)
- Prioridade (ex: Alta, Média, Baixa)
- Status (ex: Ativo, Inativo)
- Projeto (ex: Projeto X, Projeto Y)

## Dicas de Uso

- Crie um padrão de nomenclatura
- Use cores para facilitar identificação
- Mantenha tags relevantes
- Evite criar tags redundantes

:::tip
Use tags consistentes em toda a equipe para facilitar a busca e organização.
:::
